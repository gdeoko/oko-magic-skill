# Законы разнообразия (читать ПЕРВЫМИ, до сборки ролика)

«Анимация» = любой слой поверх стокового кадра: GLB/3D, 3D-графика, инфографика кодом,
кинетика, callout, штампы, партиклы, свет, шейдерные переходы, оверлеи.

## Закон 1 — всё под сценарий, всё заново
Каждый ролик анализируется отдельно (ниша, тон, метафоры, цифры, боли, оффер). Из анализа
рождаются: кадры, анимации, инфографика, беспоук-графика, музыка, SFX, грейд, обложка.
- Инфографику и беспоук-графику ПИШЕМ ЗАНОВО под конкретную цифру/метафору. Запрещено
  брать прошлую графику и менять только цифры/буквы.
- Ассеты (Lottie, gl-переходы, музыка, звук, кадры) ИЩЕМ/СКАЧИВАЕМ заново под ключевые
  слова этого сценария.
- Неизменны только бренд-константы (BRAND_PROFILE): голос, шрифт+цвет субтитров, лого,
  финалка, палитра. Озвучка/текст/сценарий/анимации/кадры/музыка — всегда разные.

## Закон 2 — уникальные кадры, темп 3–5 сек
- 10–14 УНИКАЛЬНЫХ клипов на ролик. Ни один клип не повторяется внутри ролика.
- 1 бит сценария → 1 запрос → 1 клип. Последовательность логичная: хук → проблема →
  процесс → результат/цифры → CTA.
- Между роликами кадры не повторяются (дедуп по video id, реестр USED_FOOTAGE).
- ≥2 бренд-кадра в каждый ролик (камера/съёмка/монтаж/студия), даже если тема чужая.

## Закон 3 — память запрещает повторы
- Перед сборкой прочитать USED_FOOTAGE.md и USED_ANIM.md, исключить занятое.
- Один приём/ассет — не чаще 1 раза в 3–5 роликов. Переходы и финалы всегда новые.
- После сборки дописать использованное в оба реестра.

## Мини-процедура на каждый ролик
1. Манифест: разложить сценарий на биты → запросы кадров + план анимаций + цифры→графика +
   метафоры→беспоук + 3D + музыка/SFX + промпт обложки.
2. Сверить с реестрами, заменить всё, что уже было.
3. Скачать/сгенерить/написать всё новое.
4. Собрать (3 этапа ffmpeg), плотно разложить анимации по времени (не внахлёст).
5. QC-чеклист (см. SKILL.md §9). 
6. Дописать реестры.

## Закон 4 — УНИКАЛЬНАЯ ФОРМА каждой вставки + НОЛЬ СТАТИКИ (см. MOTION_ARSENAL.md)
- Каждая вставка (сток/графика/3D) — в СВОЁМ формате/форме и не повторяет соседей.
  Библиотека форм: круг, овал, шестиугольник, пятиугольник, ромб, скруг.квадрат,
  прямоугольник, арка, наклон, TV, параллелограмм, верт./гор. полоса, телефон-мокап,
  фулл-экран, крупно-центр-с-растворением. Формы во всём ролике стараемся не повторять.
- 3 оси без повтора подряд: тип графики × раскладка(place) × форма(fmt).
- **МИКС:** часть стоков — ФУЛЛ-ЭКРАН (пан/зум), часть — в формах. Не всё в формах,
  не всё фулл-скрин, не всё «окно справа».
- **НИКАКОЙ СТАТИКИ:** вставки — только ВИДЕО (не фотки); 3D крутится всю сцену; у любой
  вставки есть движение (видео/ken-burns/пан crop-t/вращение/анимация). Замерло = брак.
- **Лицо не перекрываем** (центр → размыть его; сбоку → отодвинуть reframe). Контраст —
  затемнением видео под графикой, а НЕ квадратной подложкой (это глюк).
- Фулл-экран движение только crop-pan (zoompan zoom-out/pan даёт чёрный кадр).

## Красные флаги (значит нарушил закон)
- В двух роликах подряд один и тот же набор оверлеев/переходов/3D-фигуры.
- В ролике < 8 уникальных клипов или клип повторяется.
- Инфографика та же, поменяны только цифры.
- Обложка собрана из стока+PIL, а не сгенерирована нейросетью.
- Нет ни одного бренд-кадра.
