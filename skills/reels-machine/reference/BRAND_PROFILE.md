## ШАБЛОН — ЗАПОЛНИТЬ ПОД КЛИЕНТА (замени <…>)
═══════════════════════════════════════════════════════════════════════

- **Бренд/бейдж:** <название>
- **Ниша:** <сфера бизнеса, город, соцсеть>
- **CTA:** <«напишите … в директ» / оффер>
- **Бренд-строка финалки:** <например «услуга · город»>

### Палитра (бренд-константа) — по логотипу клиента
- Фон/тёмный: `#0d0d0d` (или фирменный тёмный)
- **Акцент №1 (главный):** `<#HEX из лого>` — идёт в оверлеи/штампы/счётчики/субтитры.
- Акцент №2: `<#HEX или белый>`
- Запрещённые цвета: всё, чего нет в бренде (НЕ ставить «дефолтный» цвет по привычке).

### Типографика
- Субтитры-караоке: `<шрифт, по умолч. Союз Гротеск Bold>`, 76, строчные. Активное слово —
  **акцент №1** (ASS-код `<&HBBGGRR&>`), прошлые белые. НЕ менять между роликами.
- Цифры/заголовки в графике: Montserrat Black / Golos 900 / Manrope 800.

### Голос (бренд-константа)
- `<ru-RU-DmitryNeural (муж) | ru-RU-SvetlanaNeural (жен)>`, rate `+8%`, boundary `WordBoundary`.

### Лого и обложка
- `logo_hd.png` — даёт КЛИЕНТ (PNG с прозрачностью). В скилл не входит.
- Обложка = ИИ-генерация (Nano Banana Pro) в бренд-цветах + **композит реального лого**.
  Если Higgsfield Ultra нет — фолбэк: стоковый кадр + текст (PIL), лого композитом.

### Бренд-кадры (вплетать ≥2 в каждый ролик)
- Ключевые слова стока под нишу клиента: `<напр. для кофейни: barista pour, latte art,
  coffee beans roasting, cozy cafe interior>`. Это «сшивает» ролик с брендом.

### ASS-цвет из HEX (шпаргалка)
HEX `#RRGGBB` → ASS `&HBBGGRR&` (порядок байт обратный). Пример: `#EA5920` → `&H2059EA&`.
